# MOCK CONSOLIDATION MIGRATION GUIDE\n\n## Overview\nAll duplicate mock implementations have been consolidated to use the SSOT MockFactory.\nThis ensures consistent behavior and eliminates maintenance overhead.\n\n## New Usage Pattern\n```python\nfrom test_framework.ssot.mocks import get_mock_factory\n\ndef test_example():\n    factory = get_mock_factory()\n    \n    # Instead of: MockAgent()\n    agent_mock = factory.create_agent_mock()\n    \n    # Instead of: MockWebSocket()\n    websocket_mock = factory.create_websocket_connection_mock()\n    \n    # Instead of: MockServiceManager()\n    service_mock = factory.create_service_manager_mock()\n```\n\n## Migration Mappings\n\n### MockAgent\n**Old:** `MockAgent()`\n**New:** `get_mock_factory().create_agent_mock()`\n\n###
